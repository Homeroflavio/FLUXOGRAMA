# FLUXOGRAMAS 
Fluxogramas são representações gráficas de processos ou algoritmos, utilizando símbolos padronizados como círculos, quadrados e setas para ilustrar o fluxo de informações ou ações.
Eles ajudam a visualizar etapas, decisões e a sequência de atividades, facilitando a compreensão de sistemas e a comunicação entre equipes. Ideal para planejar, documentar e otimizar processos.
